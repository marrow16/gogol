# GoGol GUI Shortcuts

Shortcuts enable you configure actions when a specific alt key is pressed.

Note: on Mac, the alt key is `⌥`

## Actions

| action                 | description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `boundary-mode:mode`   | Sets the current grid boundary _mode_ (`Dead`\|`Alive`)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `clear`                | Clears the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `export`               | Exports the current grid as RLE                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| `grid-height:n`        | Sets the current grid height to _n_ (2-1000)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `grid-size:wXh`        | Sets the current grid size to _w_ (width) X _h_ (height)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `grid-width`           | Sets the current grid width to _n_ (2-1000)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `heat-map`             | Turns on the heat map instrument (`Activity`)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `heat-map:type`        | Turns on the heat map instrument with _type_ (`Activity`\|`Occupancy`\|`Births`\|`Freshness`\|`Phase Parity`\|`All`)<br>_(other values will turn heat mapper off)_                                                                                                                                                                                                                                                                                                                                                                                |
| `heat-map-save`        | Saves the current heat map as a .png                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `name:format`          | Sets the name for output files (e.g. exports, heat map images)<br>The name can be any string that is a valid filename/filepath - plus any of the following format tokens:<br>* `%rule` e.g. transposed to `B2/S23`<br>* `%born` e.g. transposed to `B2`<br>* `%survives` e.g. transposed to `S23`<br>* `%perm` e.g. transposed to `4108`<br>* `%rand` transposed to current randomization setting<br>* `%now` transposed to a date/time stamp<br>* `%r` transposed to repeat iteration(s)<br>* `%R` transposed to last (current) repeat iteration |
| `random-changes`       | Applies random changes (noise) to the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `randomization++`      | Increments the randomization setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `randomization--`      | Decrements the randomization setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `randomization:n`      | Sets the randomization setting to _n_ (0-100)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `randomize`            | Randomize the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `record`               | Turns the record instrument on                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| `record:1\|0`          | Turns the record instrument on or off                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `repeat-detect`        | Turns the repeat detect instrument on                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `repeat-detect:1\|0`   | Turns the repeat detect instrument on or off                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `repeat-detect-save`   | Saves the current repeat detection (if enabled) as a report json                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `repeat:n,...,...`     | Repeats the following actions (comma separated) _n_ times<br>Example:<br>`repeat:100,randomize,step-ahead-by:500,heat-map-save`<br>Repeats can also be 'nested', example:<br>`repeat:5,rule-perm++,randomize,heat-map:Activity,repeat:10,step-ahead-by:10,heat-map-save`                                                                                                                                                                                                                                                                          |
| `rule-born-with:x`     | Sets or adjusts the current life rule born with - where _x_ can be examples:<br>* `23` sets to `B23`<br>* `\|23` ORs with 2 and 3 (effectively turning just those on)<br>* `&23` ANDs with 2 and 3<br>* `!23` flips 2 and 3 <br>* `!` flips all - e.g. `B23` becomes `B0145678`<br>_(Note: the current survives with, `S`, is unchanged)_                                                                                                                                                                                                         |
| `rule-born-with++`     | Increments the current life rule born with permutation (high order 9 bits)                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `rule-born-with--`     | Decrements the current life rule born with permutation (high order 9 bits)                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `rule-perm++`          | Increments the current life rule permutation                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `rule-perm--`          | Decrements the current life rule permutation                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `rule-perm:n`          | Sets the current life rule permutation to _n_ (0-262143)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `rule-survives-with:x` | Sets or adjusts the current life rule survives with - where _x_ can be examples:<br>* `23` sets to `S23`<br>* `\|23` ORs with 2 and 3 (effectively turning just those on)<br>* `&23` ANDs with 2 and 3<br>* `!23` flips 2 and 3 <br>* `!` flips all - e.g. `S23` becomes `S0145678`<br>_(Note: the current born with, `B`, is unchanged)_                                                                                                                                                                                                         |
| `rule-survives-with++` | Increments the current life rule survives with permutation (low order 9 bits)                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `rule-survives-with--` | Decrements the current life rule survives with permutation (low order 9 bits)                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `rule:name\|rle`       | Sets the current life rule (by _name_ or specified _rle_, e.g. `B2/S23`)                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `run-recipe`           | Runs the currently selected grid recipe                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `run-recipe:filename`  | Runs the grid recipe for the specified _filename_                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `run`                  | Runs simulation on the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `sleep:ms`             | Sleeps for a given _ms_ (millisecond) period                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `snapshot`             | Snapshots the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `step-ahead++`         | Increments the current step ahead setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `step-ahead--`         | Decrements the current step ahead setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `step-ahead`           | Performs a step ahead (by current step ahead setting)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `step-ahead:n`         | Sets the current step ahead setting to _n_ (1-9999)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `step-ahead-by:n`      | Performs a step ahead by _n_ (1-9999)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `step-delay++`         | Increments the current step delay setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `step-delay--`         | Decrements the current step delay setting                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `step-delay:ms`        | Sets the current step delay to _ms_ (0-2000)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `step`                 | Performs a single simulation step on the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `stop`                 | Stops simulation on the current grid                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `undo-to-snapshot`     | Restores grid to last snapshot                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| `wrap-mode:mode`       | Sets the current grid wrap _mode_ (`none`\|`horizontal`\|`vertical`\|`all`/`toroidal`)                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

In the shortcuts editor, multiple actions can be composed - e.g.

```
grid-size:100x100
rule:Standard
repeat:5,rule-perm++,randomize,heat-map:Activity,repeat:10,step-ahead-by:10,heat-map-save
```
